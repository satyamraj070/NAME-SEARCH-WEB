Free Fire Name Search

Search for Free Fire players by nickname and region

API Endpoint: https://satya-name-search.vercel.app/search?nickname=NAME®ion=REGION

How to Use

1 Direct API Usage

You can use the API directly by making a GET request to:

https://satya-name-search.vercel.app/search?nickname=PLAYER_NAME®ion=REGION_CODE

Replace PLAYER_NAME with the player's nickname and REGION_CODE with one of the region codes below.

2 Region Codes

Use one of these region codes:

  • IND - India
  • BR - Brazil
  • US - United States
  • SAC - South America Central
  • NA - North America
  • OTHER - Other Regions

3 Example API Calls

Search for a player named "ProGamer" in Brazil:

https://satya-name-search.vercel.app/search?nickname=ProGamer®ion=BR

Search for a player named "John" in India:

https://satya-name-search.vercel.app/search?nickname=John®ion=IND

4 Response Format

The API returns JSON data with player information:

{ "players": [ { "accountId": "1234567890", "nickname": "ProGamer", "region": "BR", "level": 45, "lastLogin": 1678901234 } ] }
