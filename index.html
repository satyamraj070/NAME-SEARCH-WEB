Bot Commands

Music Playback

/play [song]
Play a song from YouTube, Spotify, or SoundCloud
/playlist
Play entire playlist from a link
/radio
Start a live radio stream
/search [query]
Search for a song and select from results

Playback Controls

/pause
Pause the current playback
/resume
Resume the paused playback
/skip
Skip to the next song in queue
/stop
Stop the playback and clear the queue

Queue Management

/queue
Show the current music queue
/shuffle
Shuffle the current queue
/move [position]
Move a song to a different position in queue
/remove [position]
Remove a song from the queue

Additional Features

/lyrics [song]
Get lyrics for the current song or specified song
/video [song]
Get the video for the current song
/effects
Apply audio effects like bass boost, echo, etc.
/settings
Change bot settings for your chat

How to Setup

1

Find the Bot

Search for @UltimateMusicBot on Telegram or click the start button below.

2

Start the Bot

Open the chat with the bot and click the "Start" button to initialize it.

3

Add to Group

Add the bot to your group and give it necessary admin permissions.

4

Start Playing

Use commands like /play to start enjoying music in your group voice chat.
